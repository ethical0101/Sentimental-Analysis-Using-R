🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

ISSUE: Package installation fails
FIX: Run install_packages.R with admin privileges
     Try: install.packages("package_name", dependencies = TRUE)

ISSUE: "Cannot open file" error
FIX: Check working directory with getwd()
     Set correct path: setwd("path/to/project")

ISSUE: Dashboard won't load
FIX: Check if shiny is installed: library(shiny)
     Try different port: options(shiny.port = 8888)

ISSUE: Out of memory error
FIX: Increase limit: memory.limit(size = 8000)
     Process smaller datasets for testing

ISSUE: UTF-8 encoding problems
FIX: Read CSV with: locale = locale(encoding = "UTF-8")
